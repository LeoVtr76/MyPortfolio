Automate - Script Bash

Ce script Bash, créé durant mes études au CESI, compile et téléverse du code C vers des microcontrôleurs AVR ATmega328P en rassemblant tous les fichiers C d'un dossier spécifié. Il commence par définir le répertoire de travail selon l'entrée de l'utilisateur, prépare l'environnement en s'assurant que les dossiers nécessaires existent, compile chaque fichier C en fichiers objet, crée un exécutable par liaison de ces fichiers, le convertit en fichier HEX prêt pour le téléversement et propose de démarrer ce processus.
